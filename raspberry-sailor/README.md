# `raspberry-sailor`

### Utilities for Navigation and Sailing, in Java

- NMEA Parser and related utilities
- Tide Engine
- NMEA Multiplexer
- Deviation Tool (build your own deviation curve)
- Polar Smoother (elaborate your own polars)
- and much more, keep posted.
